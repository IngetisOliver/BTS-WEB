Développeur Java

Définitions :

Le développeur Java (code ROME M1805) est un informaticien capable de créer un programme informatique en langage Java. Technicien informatique, il est capable selon le cahier des charges fourni d'analyser très précisément les besoins de son client, de concevoir un logiciel d'application informatique qui répond point par point aux exigences techniques et fonctionnelles du cahier des charges, puis d'écrire en langage Java tout ou partie du programme informatique. L'informatisation des moyens et des outils de travail des entreprises, des services publics et privés a conduit à l'accroissement du nombre de développeurs informatiques, véritables experts professionnels des différents langages informatiques. Le développeur Java est un concepteur sur mesure de programmes informatiques en langage Java ainsi que de leurs notices d'utilisation. Ainsi, son travail se déroule en 3 étapes : l'analyse du cahier des charges de son client au cours de laquelle il va dégager une bonne solution technique. Ensuite, celui-ci passe à la programmation en langage Java d'une partie ou de la totalité du logiciel informatique en assemblant des programmes Java existants avant de passer à la phase d'essais pour tester les applications. Enfin, le développeur Java rédige les notices techniques et les principaux guides d'utilisation pour les usagers.
